## Technologies We Used
- React.js
- TypeScript
- Tailwind CSS
- JavaScript
- Browser LocalStorage
- npm and JSON Server
- MongoDB 
- MongoDB ATLAS

## Features
- SignIn and SignUp functionality.
- Navbar for easy navigation between pages.
- Nutrition page
- Workout Page
- Analytic Page
- EducationPage
- Calculate BMI
